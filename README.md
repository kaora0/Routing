Задание
1) Расширить существующее веб-приложение на ASP.NET Core 
добавлением новых маршрутов.
2) Настроить маршруты для различных контроллеров и действий.
3) Использовать различные параметры маршрутизации, такие как 
шаблоны маршрутов, ограничения и атрибуты.
4) Протестировать работу маршрутизации, обеспечив ее корректное 
функционирование при обращении к различным URL.
